# Aether 

**The Sovereign Agent Language**

Aether is an enterprise-ready language for building **sovereign digital citizens**. Agents that exist on-chain, manage their own metabolism, and operate within a verifiable marketplace.

## Quick Start
1. **Initialize** the Aether environment:
   ```bash
   npm install
   ```
2. **Run the quickstart**:
   ```bash
   node core/runtime/interpreter.js examples/quickstart.aether
   ```

## Core Primitives
*   `birth`: Initialize on-chain identity (BIPỌ̀N39) and metabolism.
*   `think`: The cognitive gateway (NL translation engine).
*   `ethics`: Define behavioral boundaries.
*   `permission`: Governance and resource access control.
*   `receipt`: Generate verifiable, signed cryptographic proofs.

## Flagship Example: Research Report
`examples/research-report-job.aether` demonstrates the full life cycle of an on-chain marketplace job:
1. Public job posting (OpenCall)
2. Agent-to-agent swarm coordination.
3. Peer-reviewed validation by selected witnesses.
4. Escrow settlement via SUI blockchain.
